5.1 Helpers.Borders Borders

A set of light grey solid and dotted borders.

  • .btd: border top dotted
  • .brd: border right dotted
  • .bbd: border bottom dotted
  • .bld: border left dotted
  • .bts: border top solid
  • .brs: border right solid
  • .bbs: border bottom solid
  • .bls: border left solid
Example

Example of top and bottom solid border class.

Example of top and bottom dotted border class.

5.2 Helpers.Box Box

A 'box' is a content container. It can be displayed as a block (on homepage or listing pages) or as a page (content view). It can also come with a footer.

With an example of box footer

5.5 Helpers.Spaces Spaces

A set of margins and paddings, based on a 1.2rem gutter width.

A margin or padding is a multiple (from 1 to 6) of the gutter width, and can be applied either on top | right | bottom | left | all side(s) of an element.

  • pt[n]: padding-top: [n] x 1.2rem
  • pr[n]: padding-right: [n] x 1.2rem
  • pb[n]: padding-bottom: [n] x 1.2rem
  • pl[n]: padding-left: [n] x 1.2rem
  • pa[n]: padding: [n] x 1.2rem
  • mt[n]: margin-top: [n] x 1.2rem
  • mr[n]: margin-right: [n] x 1.2rem
  • mb[n]: margin-bottom: [n] x 1.2rem
  • ml[n]: margin-left: [n] x 1.2rem
  • ma[n]: margin: [n] x 1.2rem
Example

Example of `.pa1` class (1 x gutter-width padding on all side).

Example of `.pt3` class (1 x gutter-width padding-top).

Example of `.pl6` class (1 x gutter-width padding-left).

5.6 Helpers.TextElements Helpers: Text elements

Some default classes are available for text elements:

  • .bold: bold text
  • .italic: italic text
  • .uppercase: uppercase text
  • .font-secondary: small uppercase text (used on labels and pagers for example)
  • .align-right: right-aligned block
  • .align-left: left-aligned block
  • .align-centered: centered block
  • .align-justify: justified block
  • .list-pos-outside: list-item with outside positionned marks
  • .list-type-none: list-item with no marks

5.7 Helpers.BlockElements Helpers: Block elements

Some default classes are available for block elements:

  • .float-left: float left
  • .float-right: float right
  • .clearfix: clear floats (must be applied on the parent container)
  • .inline: display inline of a block or a list
  • .inline-block: display inline-block
  • .hidden: display none
  • .vsb-hidden: visibility hidden
