3.5.3 Forms widgets

Source

_modules/_form-widget.styl

Description

Colette provides styles for:

  • Fields descriptions (with valid / invalid color indication)
  • Valid / Invalid fields

Note: you can also use Alerts components to handle success or errors messages.

Example

Field description/help text

  • Invalid format
  • This field is required

This field is valid

Markup

click to expand

<form xmlns="http://www.w3.org/1999/html">
    <div class="clearfix">
        <div class="form-field-widget col-md-4 pb2">
            <p>
                <label for="text_11">Text</label>
                <input type="text" id="text_11" aria-describedby="text_11_desc" />
            </p>
            <p id="text_11_desc" class="form-help">Field description/help text</p>
        </div>
        <div class="form-field-widget col-md-4 pb2">
            <p>
                <label for="text_12">Required text <abbr class="required" title="(ce champ est obligatoire)">*</abbr>
</label>
                <input class="form-field-error" type="text" id="text_12" required aria-describedby="text_12_desc" />
            </p>
            <ul>
                <li id="text_12_desc" class="form-help form-help-error" role="alert">Invalid format</li>
                <li id="text_12_desc" class="form-help form-help-error" role="alert">This field is required</li>
            </ul>
        </div>
        <div class="form-field-widget col-md-4 pb2">
            <p>
                <label for="text_13">Text</label>
                <input class="form-field-success" type="text" id="text_13" value="lorem ipsum" aria-describedby="text_13_desc" />
            </p>
            <p id="text_13_desc" class="form-help form-help-success">This field is valid</p>
        </div>
    </div>
</form>

3.8 Media

Source

_modules/_media.styl

Description

A media is either an image or a video, displayed with caption and/or credit mention.

  • Video: videos are made responsive, on the basis of a 16/9 ratio
  • Empty media: if the media (image or video) is not available for any reason, the block display a default N/A image instead

Example

Image

Video

Empty media

Markup

click to expand

<p>Image</p>
<figure class="media">
    <img src="http://img.20mn.fr/gmiJ0COJRYqUQM6w1jeBlw/648x415_20mn-11765.jpg" alt="This is an image">
    <figcation>This is the image caption</figcation>
    <p class="media-credit">This is the credit info</p>
</figure>
<p>Video</p>
<figure class="media media-video-container">
    <iframe width="640" height="360" src="https://www.youtube.com/embed/HwCnti9SW7o" frameborder="0" allowfullscreen></iframe>
    <figcation>This is the image caption</figcation>
    <p class="media-credit">This is the credit info</p>
</figure>
<p>Empty media</p>
<figure class="media">
    <span class="media-blank"></span>
    <figcation>This is the image caption</figcation>
    <p class="media-credit">This is the credit info</p>
</figure>
